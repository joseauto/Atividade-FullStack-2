# Atividade-FullStack-2

# TechForge - Escola de Robótica 

[cite_start]Este projeto é um mini-site institucional simples, moderno e responsivo, desenvolvido como exercício prático para a disciplina de **Desenvolvimento Fullstack**. 

[cite_start]**Instituição:** Universidade Senai Cimatec 
[cite_start]**Curso:** Engenharia da Computação 
[cite_start]**Professor:** Celso Barreto 

## Objetivo do Projeto
[cite_start]O objetivo foi criar a presença online rápida e organizada para uma escola ou pequeno negócio fictício [cite: 6, 7][cite_start], aplicando conceitos fundamentais de estruturação web, estilização e versionamento.

## Funcionalidades e Estrutura
[cite_start]O site é composto por três páginas principais[cite: 19]:
* [cite_start]**Home (`index.html`):** Apresentação da escola, chamada para ação (CTA) e cards com os principais benefícios
* [cite_start]**Sobre (`sobre.html`):** História da TechForge e nossa Missão, Visão e Valores
* [cite_start]**Contato (`contato.html`):** Formulário de comunicação e links de contato direto (redes sociais, email e telefone)

## Design e Estilização
[cite_start]A interface foi construída seguindo os requisitos do projeto:
* [cite_start]Layout responsivo, adaptando-se perfeitamente a dispositivos móveis (cards empilhados)
* [cite_start]Layout centralizado com largura máxima definida.
* [cite_start]Efeitos de transição fluidos (*hover*) nos menus e botões.
* [cite_start]Cards com bordas arredondadas e sombras para um visual moderno.

## Tecnologias Utilizadas
* [cite_start]**HTML5:** Estruturação semântica das páginas.
* [cite_start]**CSS3:** Estilização com variáveis, Flexbox e Media Queries.
* [cite_start]**Git e GitHub:** Controle de versão e hospedagem do código com commits organizados.
* [cite_start]**Vercel:** Deploy e hospedagem do site[cite: 11, 15, 63, 64].
