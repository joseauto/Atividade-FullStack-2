# Atividade-FullStack-2

# TechForge - Escola de Robótica 

Este projeto é um mini-site institucional simples, moderno e responsivo, desenvolvido como exercício prático para a disciplina de **Desenvolvimento Fullstack**. 

**Instituição:** Universidade Senai Cimatec 
**Curso:** Engenharia da Computação 
**Professor:** Celso Barreto 

## Objetivo do Projeto
O objetivo foi criar a presença online rápida e organizada para uma escola ou pequeno negócio fictício [cite: 6, 7][cite_start], aplicando conceitos fundamentais de estruturação web, estilização e versionamento.

## Funcionalidades e Estrutura
O site é composto por três páginas principais[cite: 19]:
* **Home (`index.html`):** Apresentação da escola, chamada para ação (CTA) e cards com os principais benefícios
* **Sobre (`sobre.html`):** História da TechForge e nossa Missão, Visão e Valores
* **Contato (`contato.html`):** Formulário de comunicação e links de contato direto (redes sociais, email e telefone)

## Design e Estilização
A interface foi construída seguindo os requisitos do projeto:
* Layout responsivo, adaptando-se perfeitamente a dispositivos móveis (cards empilhados)
* Layout centralizado com largura máxima definida.
* Efeitos de transição fluidos (*hover*) nos menus e botões.
* Cards com bordas arredondadas e sombras para um visual moderno.

## Tecnologias Utilizadas
* **HTML5:** Estruturação semântica das páginas.
* **CSS3:** Estilização com variáveis, Flexbox e Media Queries.
* **Git e GitHub:** Controle de versão e hospedagem do código com commits organizados.
* **Vercel:** Deploy e hospedagem do site[cite: 11, 15, 63, 64].
